# Recommendation_with_IBM

## Introduction  
In this project, you will find different types of recommender system to recommend
articles to both new and returning users, based on real data from the IBM watson
studio platform.  

## Getting Started  
* python 3.6+
* pandas
* numpy
* matplotlib
* pickle
